Feature 1

Automatically Recognized Menu Bar

The navigation bar will dynamically highlight the current webpage name, distinguishing it from others. Through JavaScript, hovering over other page names triggers highlighting, and returning to the initial state upon hover exit.

Credit Card Validation

Based on the type of credit card being chosen, the validate for each credit card number will be change. Limation of 15 digits is for American Express and 16 digits for VISA and Mastercard

Same Address Button

Clicking the "Same Address" button will automatically populate the billing address with the delivery address if filled. Otherwise, it prompts an alert for missing delivery information, ensuring seamless order processing and customer satisfaction.
